# Telnyx Frontend Design Skill — Before & After

Side-by-side comparison demonstrating the impact of the `telnyx-frontend-design` skill on AI-generated UIs.

## Structure

```
without-skill/     # AI-generated without any design guidance
  dashboard.html   # Voice dashboard — gradients, emojis, Poppins
  chat.html        # Chat app — purple theme, emoji avatars

with-skill/        # Same apps, built using the telnyx-frontend-design skill
  dashboard.html   # Voice dashboard — Telnyx tokens, Inter, semantic colours
  chat.html        # Chat app — cream palette, proper status badges
```

## How to View

Open any HTML file directly in a browser — no build step needed.
